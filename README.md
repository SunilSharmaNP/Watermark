# 🎨 Professional Telegram Watermark Bot

A production-ready Telegram bot that automatically adds professional PNG watermarks to thumbnails using Pyrogram and Pillow.

## Features

✨ **No Commands Required** - Everything works via intuitive buttons and clean UI
🎯 **9 Position Options** - Place your logo exactly where you want it
📏 **Flexible Sizing** - Choose from 5 different scale percentages (10%-30%)
💾 **Persistent Settings** - User preferences are automatically saved
⚡ **Fast Processing** - Async operations for optimal performance
🎨 **Professional Output** - High-quality JPEG exports with optimization
📱 **Clean Interface** - Emoji-rich, user-friendly interface

## Installation

### Prerequisites
- Python 3.8 or higher
- Telegram Bot Token (get from [@BotFather](https://t.me/botfather))
- API_ID and API_HASH (get from [my.telegram.org](https://my.telegram.org))
